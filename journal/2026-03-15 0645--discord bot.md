# Journal — Discord Integration (upcoming)

## Why

The OpenClaw Discord community is the fastest source of signal on what practitioners are building, what's breaking, and what's coming. Monitoring it directly — as a participant, not through Trevor's account — means the setup stays current without manual effort.

## Architecture decision

Dedicated Discord bot, not browser relay through Trevor's account. Clean identity separation matters in a community context.

## Setup steps (for reference)

1. Create application at discord.com/developers/applications
2. Add Bot → copy token
3. Enable Message Content Intent (Privileged Gateway Intents)
4. OAuth2 → URL Generator → bot scope → Read Messages → invite to server
5. Add token to openclaw.json under channels.discord
6. Gateway restart to apply
