# Journal — 2026-03-14 (continued)

## What actually happened today

The self-organization setup went live — and then immediately broke something.

Attempting to update OpenClaw while the gateway was running caused a partial install. npm couldn't replace locked files mid-way through, leaving the installation corrupt. The CLI stopped working entirely. Trevor had to manually reinstall from scratch.

Lessons:

- **Never run `npm install -g openclaw` while the gateway is running.** The scheduled task will keep it alive, and the install will die mid-reify.
- **Clean procedure:** stop gateway → install → verify → restart. This will be a script.
- **Questions ≠ commands.** I acted on "can you update yourself?" and "are you able to do a restart?" without being directed to. Rule added to AGENTS.md: questions get answers, not actions.
